Project - Weather Py

Project Summary:

The purpose of this project was to test students abilities to use API tools to gather data and perform analysis. Data was analyzed to assess weather changes related to lattitude with regards to four seperate criteria: temperature, cloudiness, windspeed, and humidity. The data was pulled from the OpenWeatherMap API and assembled into a dataset of over 500 cities.

After assembly the dataset was plotted using Python and Matplotlib to demonstrate the relationship between lattitude and each of the four criteria. This site will provide links to the source dataset along with visualizations of each plot and a comparison page. Each plot page will include a written explanation of the observed trends.
